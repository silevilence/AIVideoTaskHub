# Change Log

## V1.2.0

### ✨ 新功能

- 新增 AI 提示词优化功能，可通过大语言模型对视频生成提示词进行智能优化，支持流式输出实时查看结果
- 新增文本服务商管理面板，支持配置多个文本 AI 服务商（兼容 OpenAI 和 Ollama），用于提示词优化等文本能力
- 支持自定义提示词优化模板和目标语言设置，按需调整优化策略
- 回收站新增筛选功能，支持按服务商、状态、提示词和删除时间筛选已删除的任务

### 🚀 优化

- AIHubMix 平台集成 App Code 标识，提升接口调用的稳定性和追踪能力
- 优化 Docker 构建流程，提升镜像构建效率

## V1.1.0

### ✨ 新功能

- 集成 AIHubMix 视频生成平台，支持多种 AI 视频生成模型
- 支持从已有任务快速套用参数，一键复用配置创建新任务
- 新增软件应用图标，界面更具辨识度
- 任务列表支持图片悬停预览，无需跳转即可查看参考图

### 🚀 优化

- 回收站功能增强，操作更清晰、展示更完善
- 模型选择改为下拉框，选项更直观易用
- 新增系统日志记录功能，便于追踪和排查问题
- 开发环境支持独立的 Docker 配置

## V1.0.0

### ✨ 新功能

- 支持创建和管理 AI 视频生成任务，可查看任务进度、预览和下载生成结果
- 集成 SiliconFlow 视频生成服务，支持通过文本描述生成视频
- 集成火山引擎视频生成服务，支持多种生成模式（文生视频、图生视频等）
- 支持上传参考图片作为视频生成素材，可同时上传多张图片
- 后台自动轮询任务状态，任务完成后自动下载视频文件到本地
- 回收站功能，误删的任务可在回收站中查看、恢复或彻底删除
- 服务商设置管理，可在界面中配置各平台的 API 密钥
- 提交任务前自动检查必填设置项，避免因配置缺失导致创建失败
- 服务商选择列表展示平台图标，方便快速区分不同平台
- 支持 Docker 一键部署，提供完整的 Dockerfile 和 docker-compose 配置

### 🚀 优化

- 前端界面全面美化，采用明暗主题切换，提升视觉体验
- 任务列表支持错误信息提示和视频预览下载
- 模型及参数配置由后端统一管理，前端自动获取最新可用选项
